# Sample synthetic DFIT datasets

Each CSV is a synthetic DFIT generated by `dfit.generate_dfit`, with columns:

- `time_min` - time since start of injection (minutes)
- `pressure_psi` - bottomhole pressure (psi)
- `rate_bpm` - injection rate (bbl/min); zero during falloff
- `G` - Nolte G-function (blank during injection)

The four files cover the four canonical leakoff regimes (normal, pressure-
dependent, height-recession / transverse-storage, and fracture-tip-extension).
Gauge noise of ~2 psi has been added. Ground-truth parameters are documented in
notebook 04. These files exist so the toolkit can be run end to end without any
proprietary field data.
